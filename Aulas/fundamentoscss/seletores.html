Bem-vindo!

No CSS nós possuímos os seletores por ID

  • Seletor por Tipo:Busca elementos por uma tag HTML específica.
  • Seletor por ID (#):Busca elemntos através do atributo "id".
  • Seletor por Classe (.):Busca elementos através do atributo "class".
  • Seletor Universal (*):Seleciona todos os elementos HTML.
  • Seletor por Atributo ([atrib]):Seleciona elemntos que possuem atributos específicos em nosso documento HTML e conseguimos também buscar atributos com um valor específico.
